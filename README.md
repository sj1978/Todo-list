## Getting Started

Download repository 
Open index.html in a web browser


### Setup
- Clone the REPO
- Open the terminal
- Run: npm install
- Run: npm run build
- Run: npm start

### Testing
- Open the terminal
- Run: npm install --save-dev jest
- Run: npm install --save-dev --save-exact jsdom global-jsdom
- Run: npm install --save-dev @babel/plugin-transform-modules-commonjs
- Run: npm run test
